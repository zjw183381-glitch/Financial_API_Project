\# Financial API Project



\## Project Overview



This project initializes a financial data development environment using Python. 

The goal is to connect multiple financial data APIs and verify successful authentication and data retrieval through a Python script.



\## Technology Stack



\- Python

\- Virtual Environment (venv)

\- Git

\- GitHub

\- REST APIs



\## APIs Used



\### Federal Reserve Economic Data (FRED)



Used for retrieving economic indicators and financial data.



\### Yahoo Finance



Used for accessing stock market information.



\### Alpha Vantage



Used for retrieving stock market data through API requests.
